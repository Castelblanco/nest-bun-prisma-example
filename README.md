## Description

Project mini example Nest.js and Bun.js using Prisma ORM.

## Installation

```bash
$ bun install
```

## Running the app

```bash
# development
$ bun run start

# watch mode
$ bun run start:dev

# production mode
$ bun run start:prod
```
